

The baseline code was obtained from the following Parent paper:

'[Detection of COVID-19 epidemic outbreak using machine learning](https://www.frontiersin.org/articles/10.3389/fpubh.2023.1252357/full?&utm_source=Email_to_authors_&utm_medium=Email&utm_content=T1_11.5e1_author&utm_campaign=Email_publication&field=&journalName=Frontiers_in_Public_Health&id=1252357)' published by Frontiers in Public Health.

Giphil Cho, Jeong Rye Park, Yongin Choi, Hyeonjeong Ahn, and Hyojung Lee


The novelty aspect from this code relies in five new scripts which are meant to be initialized in the following order: SIR_Feature_Extraction.py, Policy_extraction.py, enhanced_training.py, evaluate_all.py and Visualize_results.py

Many of these scripts use libraries which must be installed prior running the scripts. 
All of the necessary requirements can be found in the covid outbreak requirements.txt file. Use the following command in the project termianl in order to get them:

 pip install -r "covid outbreak requirements.txt"

 Additionally to run the novelty files use latest python version available.
